# Script para Video de Presentación - 15 segundos

## Concepto: "Explora, Conecta, Crece"

### Storyboard (15 segundos):

**Segundo 0-2**: Hook visual
- Pantalla en negro con logo "Explora" apareciendo con animación suave
- Texto: "¿Listo para hacer crecer tus redes sociales?"

**Segundo 2-5**: Demostración rápida
- Vista de la pantalla principal mostrando perfiles deslizándose (estilo Tinder)
- Mostrar gestos: swipe izquierda (X rojo), swipe derecha (flecha azul)
- Transición rápida mostrando diferentes redes sociales (Instagram, TikTok, YouTube, etc.)

**Segundo 5-9**: Valor único
- Mostrar la función de promoción (pestaña Promoción)
- Destacar: "Promociona tu perfil en Instagram, TikTok, YouTube..."
- Visual: Diferentes redes sociales con sus logos de colores brillantes

**Segundo 9-12**: Llamada a la acción
- Muestra un perfil siendo promocionado con animación
- Texto: "Miles de personas descubriendo perfiles cada día"

**Segundo 12-15**: Cierre potente
- Logo "Explora" con animación de entrada
- Texto final: "¡Únete a Explora ahora!"
- CTA: "Descubre perfiles increíbles. Promociona el tuyo."
- URL o QR code (si aplica)

## Música recomendada:
- Inicio: Energético, moderno (tipo lofi hip-hop o pop electrónico suave)
- Medio: Sube la intensidad
- Final: Nota positiva y llamativa

## Textos sugeridos (en pantalla):
1. "¿Listo para hacer crecer tus redes sociales?" (0-2s)
2. "Desliza. Descubre. Conecta." (2-5s)
3. "Promociona tu perfil" (5-9s)
4. "Únete a Explora" (12-15s)

## Colores a destacar:
- Botón rojo (siguiente)
- Botón azul (ir al enlace) 
- Botón amarillo (detalles)
- Botón verde (retroceder)
- Fondos blancos y limpios

## Herramientas recomendadas para crear el video:
1. **Canva** - Plantillas para videos promocionales
2. **Adobe Premiere Pro** - Edición profesional
3. **After Effects** - Animaciones avanzadas
4. **CapCut** - Gratis, fácil de usar
5. **Loom** - Para grabar pantallas

## Técnicas de grabación:
- Si grabas pantalla: Usa OBS Studio o QuickTime (Mac) / Xbox Game Bar (Windows)
- Resolución: 1080p mínimo (preferible 4K para redes sociales)
- Formato: MP4, ratio 9:16 (vertical) para TikTok/Instagram Stories, o 16:9 (horizontal) para YouTube
